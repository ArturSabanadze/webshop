Updates in Version 1.2
===========================

1. Admin Panel:
 - New Functions
  - get_registrations.php: Return the count of registrations, optionally filtered by date range.
  - get_products.php: Return the count of rproducts, optionally filtered by date range.
  - get_averagePrice.php: Return the average product price.
  
2. Business Logic
 - New Functions:
  - image_upload.php: handler for image file uploads (current for User Profiles).
 - Classes:
  - User class: constructs an object for "user" table, includes setters and getters method and save method aswell.
  - UserAddress class: constructs an object for "users_adresses" table, includes setters and getters method and save method aswell.
  - UserProfile class: constructs an object for "users_profiles" table, includes setters and getters method and save method aswell.
